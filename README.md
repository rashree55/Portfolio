React + Vite

This project is a personal developer portfolio built using React and Vite, providing a fast development experience with Hot Module Replacement (HMR) and modern tooling.
The setup focuses on simplicity, performance, and a clean UI without overengineering.

**Tech Stack**

•	React 18

•	TypeScript

•	Vite

•	Tailwind CSS

•	Framer Motion

•	shadcn/ui (Radix UI)

•	EmailJS
